This is a clean, responsive portfolio template built with Next.js (Pages Router) and Tailwind CSS,
configured to export a static site and deploy to GitHub Pages at: https://akaish80.github.io/portfolio

## Setup
1. Unzip the project.
2. Replace `/public/profile-placeholder.svg` with your own image if desired (400x400 recommended).
3. Update `pages/index.js` to update projects, links, and content.
4. Install dependencies: `npm install`
5. Deploy: `npm run deploy` (requires GitHub CLI `gh` installed and authenticated)

## Notes
- The `homepage` field in package.json is set for deployment under the /portfolio path.
